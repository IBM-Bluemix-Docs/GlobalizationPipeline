# 整合 {{site.data.keyword.GlobalizationPipeline_short}} 與 Maven
{: #integratingmaven}


{{site.data.keyword.GlobalizationPipeline_short}} Maven 外掛程式讓您能將 {{site.data.keyword.GlobalizationPipeline_short}} 服務與 Maven 建置相整合。

該外掛程式可以將本端檔案系統中的可翻譯資源組合上傳到 {{site.data.keyword.GlobalizationPipeline_short}} 服務實例，並將已翻譯的資源組合下載到本端檔案系統。 

如需 {{site.data.keyword.GlobalizationPipeline_short}} Maven 外掛程式的相關資訊，請參閱 [Maven Plugin User Guide](https://github.com/IBM-Cloud/gp-java-tools/blob/master/gp-maven-plugin/README.md)。
